# Refrain 项目结构

```
refrain/
├── .env                  # API 密钥配置
├── .gitignore
├── pyproject.toml        # 项目元数据和依赖
├── README.md
└── src/refrain/
    ├── cli/              # CLI 交互层
    │   └── commands/     # 子命令模块
    ├── core/             # 核心基础设施层
    │   ├── config/       # 配置管理
    │   └── llm/          # LLM 客户端
    ├── engine/           # 任务执行引擎层
    │   └── orchestrator/ # ReAct 循环编排器
    ├── skills/           # 工具系统层
    │   ├── base/         # Skill 基类
    │   ├── registry/     # 技能注册中心
    │   └── toolbox/      # 具体技能实现
    ├── utils/            # 辅助工具层
    │   ├── fs/           # 文件系统操作
    │   └── ui/           # UI 界面渲染
    └── resources/        # 静态资源层
        └── system.md     # AI 系统提示词
```

## 各层说明

- **cli/** - 命令行交互，处理用户输入和输出
- **core/** - 基础设施，包含LLM客户端和配置管理
- **engine/** - 任务编排，实现ReAct循环逻辑
- **skills/** - 工具系统，定义和管理可调用的技能/工具
- **utils/** - 工具函数，文件操作和UI渲染
- **resources/** - 静态资源，系统提示词等
