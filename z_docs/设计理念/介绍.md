# Refrain 项目介绍

## 关于 Refrain

Refrain 是一个 Python AI 代码助手 CLI 工具，基于"迭代优化、克制有度"的设计哲学。

**核心特性：**
- **Diff 透明化**：任何代码修改前，用户都能看到完整的变更对比
- **用户确认优先**：修改建议需要用户显式确认，绝不黑盒操作
- **多模型支持**：支持 OpenAI、DeepSeek 等多个 LLM 提供商
- **技能可扩展**：通过统一的 Skill 系统轻松添加新工具

## 快速开始

### 安装

```bash
git clone https://github.com/yourusername/refrain.git
cd refrain
pip install -e .
```

### 配置 API 密钥

创建 `.env` 文件并设置你的 API 密钥：

```env
OPENAI_API_KEY=sk-...
DEEPSEEK_API_KEY=sk-...
```

### 基本使用

```bash
# 查看帮助
rf --help

# 查看版本
rf version

# 列出可用模型
rf model list

# 切换模型
rf model use gpt-4o

# 分析代码文件
rf edit myfile.py --instruction "优化这个函数的性能"
```

## 项目结构

```
Refrain/
├── src/refrain/           # 主源代码目录
│   ├── cli/               # CLI 接口层
│   ├── core/              # 核心基础设施（配置、LLM）
│   ├── engine/            # 编排引擎（ReAct 循环）
│   ├── skills/            # 工具层（函数调用）
│   ├── utils/             # 工具函数
│   └── resources/         # 资源文件（提示词等）
├── tests/                 # 测试套件
├── z_docs/                # 文档
├── pyproject.toml         # 项目配置
└── README.md              # 项目 README
```
